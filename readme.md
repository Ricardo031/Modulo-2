# Módulo 2: Desarrollo Web con HTML, CSS y JavaScript

Este módulo se centra en la evolución de un sitio web personal y de mascotas, aplicando progresivamente mejores prácticas de desarrollo, organización de archivos y dinamismo mediante JavaScript.

## Estructura del Proyecto

El proyecto está dividido en tres etapas principales, correspondientes a cada semana de aprendizaje:

### 🔹 [Semana 1](file:///home/Coder/Ricardo/Html-css/Modulo-2/semana1)

- **Objetivo**: Introducción a la estructura básica de HTML5 y estilos fundamentales con CSS3.
- **Contenido**:
  - `index.html`: Página principal con navegación básica.
  - `mascota.html`: Detalle sobre mascotas.
  - `styles.css`: Estilos globales en la raíz.
  - `img/`: Recursos visuales iniciales.

### 🔹 [Semana 2](file:///home/Coder/Ricardo/Html-css/Modulo-2/semana2)

- **Objetivo**: Organización del código y separación de responsabilidades.
- **Mejoras**:
  - Los archivos CSS se mueven a una carpeta `styles/` dedicada.
  - Se incluyen múltiples hojas de estilo (`styles.css` y `styles2.css`) para una mejor modularización.
  - Ampliación de la galería de imágenes en `img/`.

### 🔹 [Semana 3](file:///home/Coder/Ricardo/Html-css/Modulo-2/semana3)

- **Objetivo**: Interactividad con JavaScript y optimización de recursos.
- **Novedades**:
  - `js/app.js`: Introducción de lógica de programación para dinamizar el sitio.
  - `styles-Copia/`: Ejemplo de optimización con archivos minificados (`.min.css`).
  - Navegación mejorada con enlaces externos (GitHub).
  - Configuración de entorno con `.vscode/settings.json`.

## Cómo visualizar el proyecto

1. Navega a la carpeta de la semana que desees revisar.
2. Abre el archivo `index.html` en tu navegador preferido.
3. Explora la evolución del diseño y la funcionalidad entre cada versión.
